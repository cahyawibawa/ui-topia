## What is ui/topia?

**ui/topia** is a curated collection of UI components designed to balance visual appeal, accessibility, and flexibility. Built using Framer Motion and TailwindCSS, this collection offers ready-to-use designs that are easy to integrate and customize for a variety of projects.

<Callout type="info">
   The project is in beta. The components' code and styles are subject to change as improvements are made. Feedback and contributions are welcome to help refine the collection
</Callout>

## Motivation

ui/topia began as a simple idea: to create a digital space where unique UI components could inspire. The goal is to provide designers and developers a central hub for finding inspiration and solutions for their projects. This library  is heavily inspired by [shadcn/ui](https://github.com/shadcn/ui) and button collections from [ibelick/buttons](https://github.com/ibelick/buttons).
