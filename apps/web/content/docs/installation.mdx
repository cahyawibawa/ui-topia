This project is built with Next.js and leverages TypeScript and Tailwind CSS for all its components. Some components also require [shadcn/ui](https://ui.shadcn.com/docs/installation) and Framer Motion. While we recommend TypeScript for optimal development experience, JavaScript is also supported.

<Steps>
<Step>
### Install Next.js
<InstallTabs items={['npm', 'pnpm', 'yarn', 'bun']}>

```bash tab="npm"
npx create-next-app@latest my-app --typescript --tailwind --eslint framer-motion

```

```bash tab="pnpm"
pnpm create next-app@latest my-app --typescript --tailwind --eslint framer-motion

```

```bash tab="yarn"
yarn add @react-spring/web cobeyarn create next-app@latest my-app --typescript  framer-motion--tailwind --eslint
```

```bash tab="bun"
bunx --bun create-next-app@latest my-app --typescript --tailwind --eslint framer-motion

```

</InstallTabs>

</Step>

### Configure components.json
You will be asked a few questions to configure components.json:

```txt
Which style would you like to use? › Default
Which color would you like to use as base color? › Zinc
Do you want to use CSS variables for colors? › no / yes
```

</Step>

<Step>

### Add cn helper

I use a `cn` helper to make it easier to conditionally add Tailwind CSS classes. Here's how I define it in `lib/utils.ts`:

```ts title="lib/utils.ts"
import { clsx, type ClassValue } from 'clsx'
import { twMerge } from 'tailwind-merge'

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```

</Step>
<Step>
### App structure
Here's how I structure my Next.js apps. You can use this as a reference:

      <Files>
        <Folder name="app" defaultOpen>
          <File name="page.tsx" />
          <File name="layout.tsx" />
        </Folder>
        <Folder name="components" defaultOpen>
        <Folder name="ui" defaultOpen>
             <File name="button.tsx" />
        </Folder>
          <File name="site-header.tsx" />
          <File name="site-footer.tsx" />
        </Folder>
          <Folder name="lib" defaultOpen>
             <File name="utils.ts" />
        </Folder>
        <Folder name="styles" defaultOpen>
             <File name="globals.css" />
        </Folder>
         <File name="components.json" />
         <File name="next.config.js" />
        <File name="package.json" />
         <File name="postcss.config.js" />
        <File name="tailwind.config.ts" />
        <File name="tsconfig.json" />
      </Files>

- I place the UI components `shadcn/ui` in the `components/ui` folder.
- The rest of the components such as `<SiteHeader />` and `<SiteFooter />` are placed in the `components` folder.
- The `lib` folder contains all the utility functions. I have a `utils.ts` where I define the `cn` helper.
- The `styles` folder contains the global CSS.

</Step>
